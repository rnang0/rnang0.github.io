二：jstl核心库的常用标签

（1）导入lib包：使用jstl很显然我们需要导入它的jar包。http://archive.apache.org/dist/jakarta/taglibs/standard/binaries/”

网址下载 JSTL的安装包。jakarta-taglibs-standard-1.1.2.zip，然后将下载好的JSTL安装包 进行解压，此时，在lib目录下可以看到两个JAR文件，分别为jstl.jar和standard.jar。 其中，jstl.jar文件包含JSTL规范中定义的接口和相关类，standard.jar文件包含用于 实现JSTL的.class文件以及JSTL中5个标签库描述符文件（TLD）

（2）使用jsp里面的taglib指令来导入核心的标准库

<%@taglib prefix=”c” uri=”http://java.sun.com/jsp/jstl/core" %>
